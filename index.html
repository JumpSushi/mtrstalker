Quick Overview

MTRacker is a simple PCB design inspired by traintrackr.io for the Hong Kong Kwun Tong Line MTR metro system.

Key Components

  • LEDs: 66 addressable WS2812E LEDs
    • Chosen for small form factor while maintaining WS2812B benefits
    • Allows color customization without complex LED controllers
  • Microcontroller: ESP32-WROOM-32E
    • Built-in antenna
    • 8MB flash storage

Software Implementation

I've built a sample Python script with TKinter for train tracking, using MTR's API. You can find the implementation in the GitHub repository. Note: To run the script, you'll need to download both gui.py and api.py files.
